termux安装gcc转载

先安装golang 

pkg install golang

vim  $PREFIX/etc/apt/sources.list

编辑源加上一条

deb https://its-pointless.github.io/files/24 termux extras


:x保存退出后

wget https://its-pointless.github.io/pointless.gpg
apt-get install gnupg
apt-key add pointless.gpg
apt-get update

 当源无错误更新后
